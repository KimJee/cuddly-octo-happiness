Flixter

What is this?

A simple list app that shows the latest movies.

This app utilizes an themoviedb's API to see the latest movies in theaters. This app also uses the RecyclerView paradigm for an easier list view.

Link to Github Repo

Important Design Queries

  • How do we want to format the JSON data?
  • What is the best layout for each of these movies?

What I learned

  • Utilized an API to grab movie thumbnails, movie titles, and the movie summary.
  • Created an custom Adapter class, to leverage Android's RecyclerView which allows for easier list scrolling.
  • What is an API, and how to we convert JSONResponses into our own built-in objects.

Points of Weakness & Functionality Improvements

  • The visual interface could have margins, and use better themes and fonts.
  • The user should be able to select "tabs" where they can choose different "genres" of movies.
  • The user should be able to select a movie, and be referred to a more detailed page about the selected movie.
