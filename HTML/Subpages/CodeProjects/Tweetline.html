What is this?

Another Twitter Client

This app utilizes Twitter's API to view a user's timeline.

Link to Github Repo

Important Design Queries

  • How do we hide API tokens to keep our secrets safe?
  • What is the best way to utilize a REST API?

What I learned

  • Utilized a third-party library to utilize OAuth.
  • Tweets can be refreshed, thus making the timeline dynamic.
  • How to pass intents between activities via a parcel library.

Points of Weakness & Functionality Improvements

  • The visual UI elements could be further improved. As the background is blindingly white.
  • The user cannot click on a given twitter handle to go to their profile.
  • The user should be able to watch an embedded video, or "tap" on the display to launch the web browser.
