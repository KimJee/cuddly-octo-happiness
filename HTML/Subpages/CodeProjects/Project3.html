Simple Todo

What is this?

This is a simple todo list app.

The user can view the list of items. The user can then add and remove items from the todolist by holding the listitem. This app uses local storage for data persistence.

Link to Github Repo

Important Design Queries

  • What is the best way to store data for longterm usage?
  • What is an basic intuitive interface for a todo app?

What I learned

  • How to create a simple user interface via the Android studio's designer tool.
  • How to create basic listeners and handlers that react based off of user events.
  • Basic data persistence by saving the writing out the file to disk.

Points of Weakness & Things to Improve

  • The interface isn't the prettiest. There's just the barebones of a todolist, and nothing more.
  • User should be able to have sublists (nested lists), and should be able to "hide" lists. Perhaps include a date-time module that will remind the user through a notification.
